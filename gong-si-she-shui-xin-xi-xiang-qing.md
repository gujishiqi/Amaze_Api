### 公司涉税信息详情

---

#### 接口描述

* 根据企业名称及主题代码获取该公司的主题信息

#### 请求URL

* /api/company/event

#### 请求方式

* GET

#### 请求参数

| 参数名 | 必填 | 数据类型 | 默认值 | 说明 |
| :--- | :--- | :--- | :--- | :--- |
| name | 是 | String |  | 企业名称 |
| themeCode | 是 | String |  | 主题代码 |
| pageNo | 否 | int | 1 | 页码 |
| pageSize | 否 | int | 5 | 每页条数 |
