### 企业涉税信息数量列表

---

#### 接口描述

* 根据企业名称获取企业的各个涉税主题的涉税信息数量

#### 请求URL

* /api/company/person

#### 请求方式

* GET

#### 请求参数

| 参数名 | 必填 | 数据类型 | 说明 |
| :--- | :--- | :--- | :--- |
| name | 是 | String | 企业名称 |

#### 返回数据示例

```
{
  "code": 100,
  "ggxx": [
    {
      "高管名称": "王宝富",
      "高管职位": "监事"
    },
    {
      "高管名称": "舒宁",
      "高管职位": "监事"
    },
    {
      "高管名称": "郝清华",
      "高管职位": "监事"
    },
    {
      "高管名称": "安小予",
      "高管职位": "董事长"
    },
    {
      "高管名称": "王清远",
      "高管职位": "董事"
    }
  ]
}
```

#### 返回数据说明

| 参数名称 | 数据类型 | 说明 |
| --- | --- | --- |
| code | Long | 参照[全局返回代码](/数据词典.md) |
| ggxx | Object | 高管信息 |

#### 备注

* 如果`code`为101，或者`ggxx`为`null`则说明该公司暂时查询不到高管信息
