### 上市公司十大股东变化图

---

#### 接口描述

* 根据上市公司的股票代码获取上市公司十大股东变化数据

#### 请求URL

* /api/stock/line

#### 请求方式

* GET

#### 请求参数

| 参数名 | 必填 | 数据类型 | 说明 |
| :--- | :--- | :--- | :--- |
| code | 是 | String | 股票代码 |

#### 返回数据说明

| 参数名称 | 数据类型 | 说明 |
| --- | --- | --- |
| code | Long | 参照[全局返回代码](/数据词典.md) |
| timeList | Object | 时间节点集合 |
| nodes | Object | 节点 |
| edges | Object | 边 |

#### 备注

* 当请求参数`time`为空时，返回最近一个时间节点的投资关系图谱数据
