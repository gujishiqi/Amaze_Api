### 上市公司动态投资关系图谱

---

#### 接口描述

* 根据上市公司的股票代码获取

#### 请求URL

* /api/company/chart

#### 请求方式

* GET

#### 请求参数

| 参数名 | 必填 | 数据类型 | 说明 |
| :--- | :--- | :--- | :--- |
| name | 是 | String | 企业名称 |

#### 返回数据示例

```
{
  "code": 100,
  "nodes": [
    {
      "id": 0,
      "name": "中国市政工程西北设计研究院有限公司",
      "type": "企业"
    },
    {
      "id": 1,
      "name": "中国中建设计集团有限公司",
      "type": "企业"
    },
    {
      "id": 2,

      "type": "企业"

  ],
  "edges": [
    {
      "source": 1,

      "relation": "65.00%"
    },
    {
      "source": 2,
      "target": 0,
```

#### 返回数据说明

| 参数名称 | 数据类型 | 说明 |
| --- | --- | --- |
| code | Long | 参照[全局返回代码](/数据词典.md) |
| nodes | Object | 节点 |
| edges | Object | 边 |
