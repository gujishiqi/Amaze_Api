### 上市公司十大股东

---

#### 接口描述

* 根据企业名称获取企业的股东及出资情况

#### 请求URL

* /api/company/shareholder

#### 请求方式

* GET

#### 请求参数

| 参数名 | 必填 | 数据类型 | 说明 |
| :--- | :--- | :--- | :--- |
| code | 是 | String | 股票代码 |
